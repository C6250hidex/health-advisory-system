# HealthSync | Digital Medical Gateway

A comprehensive multi-role healthcare platform for instant medical advice and appointment management.

## 🚀 Features
- **3-Client System:** Specialized dashboards for Patients, Doctors, and Administrators.
- **AI Health Assistant:** Integrated with Google Gemini for real-time medical guidance.
- **Multi-Step Booking:** Professional multi-stage appointment flow with doctor proximity tracking.
- **Medical Records:** Secure patient profiles with persistent medical history.
- **Live Communication:** Integrated chat system between doctors and patients.
- **Admin Control:** Audit logs, doctor verification, and user management.

## 🛠 Tech Stack
- **Frontend:** React (Vite), Tailwind CSS, Lucide Icons.
- **Backend:** Node.js, Express.
- **Database:** MySQL.
- **AI:** Google Generative AI (Gemini 1.5 Flash).

## ⚙️ Setup
1. Clone the repo.
2. Run `npm install` in both `/client` and `/server`.
3. Set up your MySQL database using `/database/schema.sql`.
4. Create a `.env` file in `/server` based on `.env.example`.
5. Run `node server.js` and `npm run dev`.
